# Weather_Python_Project
A simple and user-friendly GUI Weather Application built using Python. This app allows users to check real-time weather information of any city using an API.

 Features:-
 
   🌍 Search weather by city name
   
   🌡️ Displays temperature, humidity, and weather condition
   
   🖥️ Interactive GUI interface
   
   ⚡ Real-time weather updates using API
   
   ❌ Error handling for invalid city names
   
Technologies Used:-

 * Python 🐍
 * Tkinter (for GUI)
 * Requests (for API calls)
 * Weather API (like OpenWeatherMap)
